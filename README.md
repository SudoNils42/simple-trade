# Simple Trade

**A lightweight, real-time trading simulator & portfolio manager.**  
Practice buying and selling stocks, cryptocurrencies, indices and commodities **with zero risk**. Perfect for learning, testing strategies, or just having fun with the markets!

**Link (as website or webapp)**: https://sudonils42.github.io/simple-trade/

## Features

- **Real-time prices** via WebSockets (stocks + crypto)
- **Interactive charts** powered by [Lightweight Charts](https://tradingview.github.io/lightweight-charts/)
- **Simulated trading** (Buy/Sell any quantity)
- **Portfolio tracking** with P&L, cash balance, and trade history
- **Smart market status** – detects US market hours automatically
- **Yahoo Finance fallback** for closed markets & historical data
- **Progressive Web App** (installable on phone/desktop)
- **Persistent storage** via localStorage (your portfolio survives refreshes)
- **Responsive UI** with Tailwind CSS + clean navigation

**Preloaded assets** (40+):
- Stocks: NVDA, TSLA, AAPL, AMD, PLTR, AMZN, META, MSFT, GOOGL, NFLX...
- Crypto: BTC-USD, ETH-USD, SOL-USD, BNB-USD, XRP-USD...
- Indices: S&P 500 (^GSPC), Nasdaq (^IXIC), Dow Jones (^DJI), VIX
- Commodities: Gold, Silver, Crude Oil
